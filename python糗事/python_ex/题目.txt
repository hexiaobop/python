 Python 是一种面向对象、解释型计算机程序设计语言，由 Guido van Rossum 于 1989 年
底发明，如今已经成为最受欢迎的程序设计语言之一，在 2015TIOBE 编程语言排行榜中位列
第 8 位，并呈线性增长的趋势。现请你用 Python 解决两个简单的问题：
（1）Bob 在某赛场上射击气球，每一个气球都有一个得分 X，得分各不相同。一共有 N
个气球，他的枪里面只有 k 个子弹，假设他的枪法很准，每发必中。请问他的最大得分是多
少。
输入格式
 第一行输入气球的个数 N 和子弹数量 k
 第二行输入 N 个气球的分数（都是正整数），中间用空格分隔
输出格式
 一个数，Bob 的最大得分（最大得分不会超过 109）
样例输入
10 3
32 4 23 74 1 62 3 12 14 42
样例输出
178
